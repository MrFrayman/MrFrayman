# Syed Maarif Al-Kadhim

Yeah, hi. I’m **Syed Maarif Al-Kadhim**, just a guy from Sharjah who spends too much time staring at code. Studying Software Engineering at Westford University College UK, because apparently I thought endless debugging sessions sounded fun. I build stuff—mostly web apps, some backend, and a bit of blockchain when I’m feeling extra unhinged. Not much for grand speeches, so let’s keep this quick.

---

## What I’m Doing, I Guess

- Messing around with **React**, **Vite**, and **Next.js** to make web apps that don’t crash (usually).
- Using **Supabase** and **PostgreSQL** for backend stuff that’s supposed to scale.
- Tinkering with **Solidity** and **Ethereum** because decentralized things are neat, I suppose.
- Playing with **AI**—mostly **OpenAI**, **LangChain**, and **TensorFlow**—to build things that think (or pretend to).
- Also, something about **image processing** for public transport. Don’t ask, it’s a long story.

Currently learning more about **smart contracts** and **autonomous AI systems**, but honestly, I’m just winging it half the time.
